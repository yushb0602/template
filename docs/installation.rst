============
Installation
============

.. math::
   :nowrap:

   \begin{eqnarray}
      y    & = & ax^2 + bx + c \\
      f(x) & = & x^2 + 2xy + y^2
   \end{eqnarray}
   
Since Pythagoras, we know that :math:`a^2 + b^2 = c^2`.
Install the package with pip::

    $ pip install read-the-docs-template
